# Physics Visualizer

一个用 AI 将中文物理题目自动转化为可交互 2D 仿真的工具。两天内完成。

**亮点**：AI 生成 DSL 结构校验通过率 100%（DeepSeek Flash，100 题测试）· 20 个手写场景 · 7 类力分量可视化 · 全 SI 单位

## 启动

```bash
npm install
npm run dev          # → http://127.0.0.1:5173
npm test             # 16 tests
```

## 能做什么

- 输入中文物理题，自动生成可运行的 2D 仿真
- 播放、暂停、拖拽时间轴、缩放平移
- 查看速度、加速度、受力（重力/弹力/支持力/摩擦力/电场力/洛伦兹力/运动曲线等效力）的实时数值和时序图
- 调整参数滑块（质量、刚度、恢复系数等）
- 覆盖场景：自由落体、抛体、弹簧振子、弹性/非弹性碰撞、单摆（铰链/绳/弹簧）、斜面、滑轮、匀强电场/磁场、点电荷（固定/移动）、时变/空间变化力场、事件触发控制

## 不能做什么

- 复杂多体耦合（三体以上）、热力学、流体、波动、量子
- 3D 渲染
- 物理参数自动校验——AI 生成的初始位置、质量、刚度可能不准确；结构正确 ≠ 物理正确

## 思路

没有让 AI 直接生成代码，而是设计了一个中间协议（Graph DSL）——AI 输出结构化的物理描述，Box2D 引擎执行。两者通过协议解耦。

AI 流水线分三步：先从题目提取语义中间表示（IR），再转成 DSL JSON，校验不通过则自动修复。提示词经过多轮迭代优化，生成通过率从最初的 79% 逐步提升到 100%。

比较满意的几个点：
- **场系统**：支持匀强场、径向场（含 1/r² 衰减）、时变/空间变化场，径向场原点可绑定到运动物体
- **力可视**：7 类力分量按来源分色显示，Inspector 中可查看任意物理量的时序图
- **事件系统**：支持时间/条件/碰撞触发，可控制速度/位置/冲量，自动反算等效力
- **提示词工程**：从中文到英文、从统一到分阶段、从默认值到反模式清单，系统性迭代

## 技术栈

React · TypeScript · Box2D-WASM · Canvas 2D · Vite · Vitest · DeepSeek API / GLM-5.2

## 目录

```
src/
├── graph/          # DSL 类型定义与校验
├── sim/box2d/      # Box2D 世界、刚体、关节、力、事件
├── sim/            # 仿真主循环
├── render/         # Canvas 渲染
├── services/       # AI 客户端、DSL 生成、提示词
├── components/     # React UI
schema/             # JSON Schema (1257 行)
scripts/            # 批量测试工具
examples/           # 20 个手写场景
```

## 局限

- AI 生成质量依赖于提示词和模型，复杂题仍需人工调参
- 未覆盖刚体转动、角动量等高中物理进阶内容
- 网页未适配移动端
